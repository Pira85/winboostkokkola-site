Miten tarkistat, tukeeko koneesi Windows 11:ä – vaiheittain

Käytännön ohjeet Windows 11 -yhteensopivuuden tarkistamiseen.

1. Lataa PC Health Check

Microsoft tarjoaa ilmaisen PC Health Check -työkalun. Sen avulla näet nopeasti, täyttääkö koneesi Windows 11:n laitevaatimukset.

2. Tarkista TPM 2.0

Paina Win + R → kirjoita tpm.msc. Jos näet version 2.0, koneesi täyttää tämän ehdon.

3. Tarkista Secure Boot

Avaa asetukset → Päivitys ja suojaus → Palautus → Käynnistä uudelleen nyt → UEFI-asetukset. Tarkista, että Secure Boot on käytössä.

4. Tarkista prosessorin sukupolvi

Windows 11 tukee vain tiettyjä Intel- ja AMD-sarjoja. Katso virallinen lista.

5. Mitä jos kone ei täytä vaatimuksia?

Kaikki ei ole menetetty. Voimme usein päivittää koneen silti, tai vaihtoehtoisesti asentaa esim. ChromeOS Flexin tai Linuxin, jolloin kone saa lisää käyttöikää.
